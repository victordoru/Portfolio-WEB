como hago para que en display grid siempre se rellenen los huecos si caben
ChatGPT

Para lograr que los huecos se rellenen automáticamente en un contenedor con display: grid cuando caben elementos adicionales, puedes utilizar la propiedad grid-auto-flow con el valor dense. Esto permite que los elementos se coloquen en los huecos disponibles, incluso si no siguen el orden original. Aquí tienes un ejemplo:

css
.contenedor {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));
  grid-auto-rows: 200px; /* Altura fija de las filas */
  grid-auto-flow: dense; /* Rellena huecos automáticamente */
}

.item {
  background-color: #ccc;
  border: 1px solid #000;
  padding: 10px;
}

En este ejemplo, grid-auto-flow: dense; permite que los elementos se coloquen en los huecos disponibles, incluso si no siguen el orden original de los elementos. El uso de auto-fill y minmax en grid-template-columns garantiza que las columnas se ajusten automáticamente al tamaño del contenedor y mantengan un ancho mínimo de 200px. Esto facilita que los elementos se rellenen correctamente en el diseño.
